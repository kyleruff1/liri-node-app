liri app readme

1.  app is meant to utilize dotenv, axios, spotify, and moment to retrieve song information from spotify api
2.  user input is retrieved and put into spotify search with results readout printed to the log.txt file showing history
3. To run the app, you will need to install npm node-spotify-api, dotenv, and axios to the terminal.   
After doing this, you are able to toggle the search by typing "node liri.js spotify-this-song" followed by the song name in quotations and you will reveive a readout of all matching song names from the node-spotify-api.

4. link to deployed version of app https://kyleruff1.github.io/liri-node-app/
